## Prerequisites

- Node.js v18+
- npm or yarn or pnpm

<CodeTabs>
<Steps>

<Step> 
Clone the repository

```bash
git clone https://github.com/duongductrong/shadcn-docs.git
```

</Step>

<Step> 
Install dependencies

```bash
npm install
```

</Step>

<Step>
Run the development server

```bash
npm run dev
```

</Step>
</Steps>

</CodeTabs>
